# EVMMAX

EVMMAX stands for **EVM** **M**odular **A**rithmetic e**X**tensions. It is a set of instructions enabling high performance implementation of cryptographic functions as EVM smart contracts, and without requiring so-called precompiles.

The latest work in progress specification is available in [EIP-6690](https://eips.ethereum.org/EIPS/eip-6690). For discussions, topics tagged [`evmmax` on Ethereum Magicians](https://ethereum-magicians.org/tag/evmmax) are relevant, and see this [issue tracker](https://github.com/ipsilon/evmmax/issues). Live discussions also take place on the [#evmmax](https://discord.com/channels/595666850260713488/1295427737183588363) channel on the Eth R&D Discord (invite link [here](https://ethereum.org/en/community/get-involved/#developers)).
